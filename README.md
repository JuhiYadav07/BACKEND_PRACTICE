# Backend Practice - CRUD API

## Project Description
A simple REST API built using Node.js, Express.js, and MySQL. The project performs CRUD (Create, Read, Update, Delete) operations on a books database.

## Tech Stack
- Node.js
- Express.js
- MySQL
- Postman

## Features
- Add a book
- Get all books
- Update book details
- Delete a book

## API Endpoints

GET /books

POST /books

PUT /books/:id

DELETE /books/:id

## How to Run

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
node index.js
```
